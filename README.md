# Usando o OpenRadioss

Existem duas maneiras principais de começar a usar o OpenRadioss em seu ambiente:

### Opção 1: Usar Binários Pré-Compilados  

Alternativamente, se você preferir não compilar o código, binários pré-compilados estão disponíveis para várias plataformas:

1. **Baixar Binários**
Acesse [downloads.openradioss.com.br](downloads.openradioss.com.br) e baixe a versão pré-compilada mais recente para o seu sistema operacional.

2. **Extrair e Instalar**
Siga as instruções de instalação fornecidas com os binários baixados.

### Executando Simulações

Depois de ter o OpenRadioss instalado (compilado ou via binários), você estará pronto para executar simulações:

- Siga o guia [Executando o OpenRadioss](EXECUTAR.md) para instruções sobre configuração, preparação de arquivos de entrada, seleção de parâmetros e interpretação de resultados.

### Atualizações 

- Novas versões são lançadas regularmente. Verifique [Novas Versões do OpenRadioss](VERSOES.md) para obter informações sobre as últimas atualizações e novos recursos.

### Opção 2: Compilar a partir do código-fonte

Se você precisar compilar o OpenRadioss a partir do código-fonte, siga estas etapas:

1. **Obter o Código-Fonte**
Primeiro, obtenha o código-fonte clonando o repositório oficial ou baixando o arquivo compactado, seguindo o guia [Como Obter o OpenRadioss](OBTER.md).

2. **Compilar**
Depois de obter o código, você precisará compilá-lo para gerar o executável. Este processo pode variar conforme seu sistema operacional e ambiente. O guia [Compilando o OpenRadioss](COMPILAR.md) fornece instruções detalhadas.

# Explorando os Resultados de Simulação

Após executar suas simulações com o OpenRadioss, você pode querer analisar e visualizar os resultados de maneiras diferentes. Para facilitar isso, o OpenRadioss oferece algumas ferramentas de pós-processamento práticas.

### Visualização em VTK

O formato VTK (Visualization Toolkit) é amplamente utilizado para visualização científica de dados. O OpenRadioss inclui uma ferramenta que converte os arquivos de animação gerados durante a simulação para o formato VTK, permitindo que você visualize a evolução da simulação em softwares de visualização compatíveis com VTK.

Para utilizar essa ferramenta, consulte o guia [Arquivos de Animação para VTK](https://github.com/OpenRadioss/OpenRadioss/tree/main/tools/anim_to_vtk).

### Dados em Formato CSV

Além da visualização, você pode querer analisar os dados de simulação em formato tabular. O OpenRadioss fornece uma ferramenta que converte os arquivos de histórico de tempo gerados durante a simulação para o formato CSV (Valores Separados por Vírgula).

Esse formato é amplamente compatível com várias planilhas e softwares de análise de dados, permitindo que você analise facilmente os dados numéricos da simulação.

Para utilizar essa ferramenta, consulte o guia [Arquivo de Histórico de Tempo](https://github.com/OpenRadioss/OpenRadioss/tree/main/tools/th_to_csv).

Com essas ferramentas de pós-processamento, você pode tirar o máximo proveito dos resultados de suas simulações, visualizando e analisando os dados de maneiras úteis para suas necessidades específicas de projeto e análise.
